The Longest Android Lock Screen Pattern

December 16, 2025

Android devices have you create a password by drawing a path through a 3x3 grid of evenly spaced points, but if you pass an unvisited point on the way to another point, you must visit that point too. Given this restriction, what's the longest path you can make? What happens when we generalize this to a nxn grid?
